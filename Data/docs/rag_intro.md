# RAG 系统简介

RAG（Retrieval-Augmented Generation，检索增强生成）是一种结合信息检索与大语言模型的 AI 架构。

## 工作原理

1. **文档加载**：从 PDF、TXT、Markdown、Word 等格式中提取文本
2. **文本切割**：将长文档切分为适合检索的文本片段（Chunks）
3. **向量化嵌入**：使用 Embedding 模型将文本转换为向量
4. **向量存储**：将向量存入数据库，构建索引
5. **检索生成**：用户提问时，检索最相关的文档片段，作为上下文提供给 LLM 生成答案

## 优势

- 解决 LLM 知识截止日期问题
- 降低幻觉率，提供可追溯的引用来源
- 无需微调即可接入私有数据

## 适用场景

- 企业内部知识库问答
- 产品文档智能助手
- 学术论文辅助阅读
- 法律文书分析检索

RAG 已成为企业级 LLM 应用的事实标准架构。
